# IronFetch

Reliable HTTP requests for TypeScript applications.

A lightweight, type-safe HTTP client built on top of the native Fetch API with support for retries, exponential backoff, request timeouts, and custom error handling.

---

## Features

- Type-safe request and response handling
- Automatic retries for transient failures
- Exponential backoff with jitter
- Request timeout support
- Custom `HttpError` and `TimeoutError`
- Native `fetch` API under the hood
- Zero external dependencies
